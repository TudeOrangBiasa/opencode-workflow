# Coding Style

Generated by `setup-matt-pocock-skills`. Edit to match project conventions.

## Naming
- **Variables/functions**: camelCase
- **Components/classes**: PascalCase
- **Files**: kebab-case (components) or snake_case (Python modules)

## CSS
- Framework: Tailwind / CSS Modules / styled-components / vanilla CSS
- Design tokens in DESIGN.md when available

## Imports
- Absolute paths preferred
- No barrel exports (named exports only)

## Error handling
- try/catch for sync, .catch() for promises
- Custom error classes for domain errors

## APIs
- RESTful endpoints
- JSON request/response
- Validation at trust boundaries
