⚡ Current Gaps

  1. Advisor passive — on-demand only, no automatic quality gate
  2. No in-line judge — builder output goes straight to reviewer without quality check
  3. No context drift detection — builder can drift from original scope undetected
  4. Step 4+5 overlap — verify-evidence + reviewer both check AC (redundant)
  5. No skip-reviewer criteria — every task runs reviewer even typo/docs
  6. Max 3 subagents — planner + builder + reviewer = 3. Adding advisor = 4 (breaks rule)
  7. No explore/scout in workflow steps — steps mention only builder/reviewer
