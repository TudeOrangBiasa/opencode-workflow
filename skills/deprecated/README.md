# Deprecated

Skills no longer actively used. Archived for reference only — do not load in new sessions.

## When to use

You need to reference a skill that was explicitly replaced, archived, or deemed unnecessary. Do not use for new tasks.

## Boundary with sibling buckets

**Deprecated** is for skills that were removed from active use. Unlike **In Progress** (drafts awaiting promotion), deprecated skills are finished — they will not be revived. Unlike other buckets, deprecated skills are excluded from compliance audits and not referenced in the top-level skill list. They remain available via global symlinks in case a reference is needed.

## Structure

Flat list, no sub-directories. Skills ordered by archive reason.
